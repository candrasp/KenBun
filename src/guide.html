Why is this needed?
KenBun manages your server's port from its Settings panel. However, this framework uses app.listen(port) internally (not Bun.serve), so KenBun's automatic port injection cannot override a hardcoded value. The fix is simple: read the port from an environment variable.
1

Create a .env file in your project root

Create a file named .env in the root folder of your project with this content:

PORT=3000
✅ Bun automatically loads .env — no extra library needed!
